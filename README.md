# PharmaShare
PharmaShare est une application destinée au **partage sécurisé de médicaments** entre particuliers et pharmacies locales. L’objectif est de réduire le gaspillage et faciliter l’accès aux médicaments encore valides.
## Fonctionnalités
- **Donner un médicament** : scanner le code-barres ou saisir les informations, avec vérification de la date de péremption.  
- **Recevoir un médicament** : rechercher par nom ou catégorie, consulter les détails et la disponibilité.  
- **Carte interactive** : localisation des médicaments disponibles avec coordonnées des utilisateurs et pharmacies.  
- **Messagerie et notifications** : communiquer avec les donneurs ou receveurs.
## Objectif
Offrir une solution pratique pour partager les médicaments non utilisés, améliorer la santé publique et réduire le gaspillage.
## Technologies
- Java + JavaFX (application desktop)  
- Base de données MySQL  
- Algorithmes de recherche pour trouver rapidement les médicaments
